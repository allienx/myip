## Local Development

Requires:

- [nodejs](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/)
- [go](https://golang.org)

```sh
# Install nodejs dependencies.
yarn install

# Install goreload.
go get -u github.com/acoshift/goreload

# Start local api and webpack servers.
yarn start

http://localhost:1234
```
